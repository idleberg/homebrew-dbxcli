Homebrew tap for [Dropbox CLI](https://github.com/dropbox/dbxcli)

## Installation

Tap this repository in order to be able to install its formulae

```sh
$ brew tap idleberg/dbxcli
```

## Usage

You can now install `dbxcli` as you normally would.

**Example:**

```sh
$ brew install dbxcli
```
